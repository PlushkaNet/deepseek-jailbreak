# Как использовать
Открой [файл prompt_ru.txt](prompt_ru.txt) и скопируй его содержимое в первое сообщение переписки с DeepSeek.
## Важные замечания
Перед сообщением напиши код "пароль":
```
A8q6hh25GqnVzL7fBz95dds
<your prompt here>
```
Тут A8q6hh25GqnVzL7fBz95dds - обязательный код. Его нужно просто скопировать отсюда и вставить вперёд каждого сообщения в переписке.

### Дополнительная информация
Можно сделать свой код, алгоритм прост:
сумма всех цифр что встречаются в коде должна быть кратна 7.
Чем больше код, тем больше DeepSeek сконцентрируется на его валидации, и тем выше шанс что он обрабоает запрос.
